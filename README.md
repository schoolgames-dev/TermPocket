# TermPocket
This is a terminal for developers on Android inside 1 python script

Only works on Android!

how to install:
1. download the TermPocket.py in the releases
2. go to your file manager
3. go to the /storage/emulated/0/ folder
4. create a Terminal file
5. create a notes, a scripts and a datacontainer folders inside the Terminal folder
6. download Pydroid 3 from the play store
7. click on the file and run it with pydroid

commands:
cf note -> creates a note.txt file
cf script -> creates a script.py file
cf json -> creates a data.json file
cf xml -> creates a data.xml file
